# 📊 Advance Excel With Data Analytics

A complete hands-on learning repository for mastering **Microsoft Excel** from basics to advanced data analytics techniques.

---

## 🚀 Overview

This project is designed for students and beginners who want to learn **Excel for Data Analysis** in a structured and practical way.

It covers:
- Excel formulas
- Data cleaning techniques
- Pivot Tables & dashboards
- Power Query & data transformation
- VBA automation

Each file represents a specific concept with examples.

---

## 📂 Project Structure

The files are organized topic-wise:

### 🔹 Basic to Intermediate Excel
- VLOOKUP, HLOOKUP, XLOOKUP
- Sorting & Filtering
- Find & Replace
- Freezing Panes
- Removing Duplicates
- Text to Columns

### 🔹 Advanced Formulas
- DSUM with AND/OR Criteria
- DAVERAGE, DCOUNT, DMAX, DMIN
- SUBTOTAL Function
- Data Validation & Dropdowns

### 🔹 Pivot Tables & Reporting
- Creating Pivot Tables
- Formatting Pivot Data
- Filters & Columns
- Working with Dates
- Custom Calculations

### 🔹 Power Query (Data Transformation)
- Data Import & Cleaning
- Merging & Appending Queries
- Custom Columns
- Workflow Automation

### 🔹 Power Pivot & Data Modeling
- Data Models
- Relationships
- RELATED Function

### 🔹 VBA & Automation
- Macro Recording
- VBA Basics
- Buttons & Form Controls
- Developer Mode
